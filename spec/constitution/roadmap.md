# Roadmap

## Hecho ✅

1. **001 · Landing Page** — Página principal pública con logo, nombre, eslogan y botón "Comenzar".
2. **002 · Autenticación** — Login, registro y logout con Supabase Auth (raw fetch).
3. **003 · Header + Layout** — Barra de navegación superior, modo oscuro toggle.
4. **004 · Catálogo de cursos** — Lista pública de cursos disponibles.
5. **005 · Matrícula** — Enrollarse en un curso (detalle + botón).
6. **006 · Dashboard alumno** — Cursos matriculados en `/dashboard`.
7. **007 · Admin CRUD** — Crear/editar/eliminar cursos, módulos, lecciones.
8. **008 · Subida de archivos** — MP3/PDF/MP4 a Supabase Storage + vista de lección `/clases/[id]`.
9. **009 · Admin: alumnos** — Sección "Alumnos" en el header admin con tabla de alumnos y matrículas.

## Siguiente 🔜

- **010 · Progreso de lecciones** — Marcar lección como completada.
- **011 · Perfil de usuario** — Editar nombre, apellidos, email.
