# 004 · Catálogo de cursos

## Objetivo

Mostrar una página pública con la lista de cursos disponibles para que los visitantes
puedan ver qué cursos ofrece Granade Music antes de registrarse o iniciar sesión.

## Requisitos funcionales

- La ruta `/cursos` muestra un grid de tarjetas de curso.
- Cada tarjeta contiene: nombre del curso, descripción breve e imagen.
- Los datos se obtienen de la tabla `courses` en Supabase (PostgreSQL).
- La página es pública (no requiere autenticación).
- Si no hay cursos, se muestra un mensaje vacío.
- La página es responsive (1 columna en móvil, 2 en tablet, 3+ en escritorio).
- El header muestra un enlace "Cursos" que redirige a `/cursos`.

## Criterios de aceptación

- [ ] Navegar a `/cursos` muestra el grid de cursos.
- [ ] Cada tarjeta muestra nombre, descripción e imagen.
- [ ] Los datos vienen de la base de datos.
- [ ] No requiere login.
- [ ] El enlace "Cursos" aparece en el header.
- [ ] Responsive.
- [ ] Mensaje de "sin cursos" si no hay datos.

## Modelo de datos

```sql
create table courses (
  id        bigint generated by default as identity primary key,
  name      text not null,
  slug      text not null unique,
  description text,
  image_url text,
  created_at timestamptz not null default now()
);
```
